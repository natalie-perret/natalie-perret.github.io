If you compared building a website with building a house: HTML would be the walls, floor and roof, CSS would be the wallpaper, choice of couch and art on the wall, and the DOM would be your architectural design or blueprint.

Boxifying design means breaking down the design elements of a website into boxes. The box model describes the box that wraps around each HTML element consisting of margins, borders, padding and the actual content.
