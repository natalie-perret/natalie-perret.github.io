JS Language

Parentheses ( ) are used to create a function, they don't always need to include text, but they can if you need to add a parameter. They can also be used to separate text within a bracket like in algebra - and we all love algebra...not!

Brackets [ ] are used to construct an array. You can have multiple within an array and you can then access parts of your array by calling the number of the bracket.

Braces { } (or as I call them, curly brackets) are used to construct an object (when using literal notation) and to enclose parts of your code. You definitely don't want to forget to add a closing curly bracket or you'll be scratching your head for days trying to figure out why your code doesn't work!!

Single quotes ' ' and Double quotes " " are generally used interchangeably and need to be around any text (known as "string") you are using in your code. There are a number of words that are used in JavaScript to execute stuff. So if you don't put quotation marks around your text then JavaScript might get confused.
