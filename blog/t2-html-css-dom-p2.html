Grid based design divides a website into a bunch of rows and columns to better organise content and keep elements aligned and ordered.

Responsive webpages are important because back in the day, people viewed website content on their desktop computers. Now, they view it on smartphones, tablets, and laptops which means the webpages needs to respond to the varying screen sizes and adjust content accordingly. Adaptive design allows websites to adjust content for specific devices.

Semantic structure is the use of HTML tags to reinforce the meaning of hte information to be displayed. For example if we want to disply text in italics, then we use the tag which indicates 'emphasis'.

'Media queries' changes the webpage display depending on the device being used to view the content. 'Overflow' contains text into a small area and adds a scroll bar. Use 'padding' and 'margins' to add negative space around content.
